# Precedents Markdown Schema

## Frontmatter

- `id`: precedent identifier
- `case_name`: case name
- `case_number`: docket / case number
- `court`: deciding court
- `date_judgment`: judgment date (`YYYY-MM-DD`)
- `judgment_type`: judgment type
- `holding`: short holding
- `summary`: short summary

## Body

The body contains the precedent's full text.
