# 판례: 2019다9876

## 사안
임차인이 임대인의 동의 없이 임대차목적물을 단기숙박업소로 운영함.

## 판결
- 계약위반 성립
- 임대차계약 해지 인정
- 보증금 반환 시 위약금 공제 인정

## 판시사항
주거용 임대차계약에서 주거 외 목적 사용은 임대인의 신뢰를 배반하는 중대한 계약위반에 해당.
